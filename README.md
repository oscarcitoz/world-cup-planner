# Cronograma Mundialista

Pagina estatica para mostrar el viaje completo: llegada, traslados, planes por hora, sectores y recomendaciones de comida por dia.

## Ver la pagina

Abre `index.html` en el navegador. No necesita servidor.

## Donde poner el Excel

Pon tu archivo `.xlsx` en la carpeta:

`inputs/`

Cuando este ahi, lo reviso y convierto el contenido real del Excel en el archivo `data.js`.

## Publicar en GitHub Pages

1. Sube la carpeta del proyecto al repositorio.
2. En GitHub entra a `Settings > Pages`.
3. Selecciona `Deploy from a branch`.
4. Elige la rama principal y la carpeta raiz del proyecto.
5. La pagina funcionara como sitio estatico, sin backend.
